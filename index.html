Improve graphics in your favorite games with Voodoo Shader.

The Voodoo Shader Framework extends and improves graphics in old and new games. Better yet, it allows you to use the same effects in all your games and tweak them when you want to. It's fast, stable, and comes with a simple UI. It runs on ATi/AMD and nVidia, in DirectX and OpenGL games. Voodoo Shader is free and open-source, so you can change it if you want or check out the details.

More technically, Voodoo Shader is a comprehensive graphics framework designed to power shaders across multiple programs, during development or after release. Voodoo uses a single core with plugins supporting many games and other applications.

Designed so effects can be loaded into any game, Voodoo Shader uses plugins to fine-tune each game individually for the best speed and stability possible. A simple UI helps you manage which plugins run in each game. The UI allows you to set up games, download and update plugins, and change the settings for each game. Games have a config file associated with them, which Voodoo Shader checks for when the game starts. If the config is found, Voodoo Shader loads using the settings you've chosen.

Right now, Voodoo Shader is in alpha testing. It is functional, but not complete. A public alpha test is being set up; the process will be simple: download the UI and demo package then make sure Voodoo Shader runs properly. If you are able, please get in touch (info below).
