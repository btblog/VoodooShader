void * VoodooShader::VSModuleManager::FindFunction ( _In_ const String &  module,
_In_ const String &  name 
) const [virtual]

Finds the address of a function in a module.

The module must be loaded into the process and export the symbol, otherwise this will return nullptr. If the module name does not have an extension, ".dll" will be appended during the lookup. It is recommended you do not use an absolute path here.

Parameters:
moduleThe module name to look up.
nameThe function name to find.
Returns:
The function's address if found, nullptr otherwise.
