virtual bool VoodooShader::VoodooWFS::VSWFile::Open ( _In_ FileOpenMode  mode) [virtual]

Opens the file for read-write access.

Parameters:
modeThe mode to open the file in.
Returns:
Whether the file was successfully opened.
Note:
If this file handle was returned by a VSWFileSystem and the absolute path is still valid, this function should always succeed. In single-threaded or fullscreen scenarios with few user tasks, this is usually the case.
Warning:
If this file handle uses a relative path, then it is subject to changes in working directory by the any module in the process. VSWFileSystem::GetFile() uses an absolute path in the constructor and is not susceptible to this.

Reads a chunk of data from the file.

The file must have been opened for reading previously, or this call will fail.

Parameters:
countThe number of bytes to read, -1 for all.
pBufferThe buffer to be read into (may be nullptr, see notes).
Returns:
The number of bytes read.
Note:
If
  • buffer is nullptr, the number of bytes that would have been read is returned but the file position is unchanged. If
  • count is -1, this returns the filesize remaining
