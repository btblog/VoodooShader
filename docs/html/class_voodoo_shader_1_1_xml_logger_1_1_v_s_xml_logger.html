Detailed Description

Xml logger class, writes neatly formatted logs for use with the log viewer.

Supports all Voodoo ILogger features and debug logging (with no buffer).

Warning:
If the logger isn't destroyed properly, the main tag of the log file won't be closed and the log won't be valid XML.

String VoodooShader::XmlLogger::VSXmlLogger::LogDate ( ) const [private]

Formats a date for the log.

The date will have the form YYYYMMDD. Leading zeros are guaranteed to be present, so the date length is 8 chars.

Note:
If the system time cannot be retrieved, an equal-length error stamp of 00000000 will be returned.

String VoodooShader::XmlLogger::VSXmlLogger::LogTime ( ) const [private]

Formats a timestamp for the log.

The timestamp will have the form HHMMSS. Leading zeros are guaranteed to be present, so the timestamp length is 6 chars.

Note:
If the system time cannot be retrieved, an equal-length error stamp of 000000 will be returned.
