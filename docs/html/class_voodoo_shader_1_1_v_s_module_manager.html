VoodooResult VOODOO_METHODTYPE VoodooShader::VSModuleManager::LoadFile ( const String &  filename) [virtual]

Loads a single module, using an absolute or relative filename.

Parameters:
filenameThe file to load, sent through the parser before use.
Note:
If a relative path is provided, this will canonicalize that path relative to Voodoo's global root. For example, the call will be loaded as: 
      Call: IModuleManager->LoadFile("../Library.dll");
      Absolute: Canonicalize("$(path)\\bin\\../Library.dll");
      Parsed: Canonicalize("C:\\Voodoo\\bin\\../Library.dll");
      Final: "C:\\Voodoo\\Library.dll";
Among other things, this guarantees an absolute path for use when searching for dependencies.
The module's directory is used in the search path for required DLLs. The search path, through this method, is:
  • The directory given with the filename, or Voodoo's global root if a relative path was given.
  • The target application's directory.
  • The current directory.
  • The system directory.
  • The Windows directory.
  • All directories listed in the PATH environment variable, in order.
