virtual VoodooResult VoodooShader::IParameter::AttachParameter ( IParameter *  pParam) [pure virtual]

Attaches a second parameter to this one, forcing the other to update whenever this value is changed.

Parameters:
pParamThe parameter to bind to this one.
Warning:
This cannot be used to bind one effect's parameter to another. It can only be used to bind parameters to virtual parameters.
virtual VoodooResult VoodooShader::IParameter::DetachParameter ( IParameter *  pParam) [pure virtual]

Detaches a parameter from this one.

Warning:
If called on a parameter that has other parameters bound to it (a source), it will disconnect all of those. This breaks connections from either end, source or bound.
