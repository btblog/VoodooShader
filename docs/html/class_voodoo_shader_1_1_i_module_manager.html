virtual bool VoodooShader::IModuleManager::ClassExists ( _In_ const String &  name) const [pure virtual]

Tests to see if a class exists in the list provided by all loaded modules.

This first attempts to convert the name string to a UUID (String::ToUuid()), and failing that, looks up the name in the internal map and checks for a corresponding UUID. The name lookup is imprecise, only the first-loaded class with a given name will be registered.

Parameters:
nameThe class name.

virtual _Check_return_ IObject* VoodooShader::IModuleManager::CreateObject ( _In_ const String &  name) const [pure virtual]

Create a new instance of the given class, using a class name or UUID in string form.

This first attempts to convert the name string to a UUID (String::ToUuid()), and failing that, looks up the name in the internal map and checks for a corresponding UUID. The name lookup is imprecise, only the first-loaded class with a given name will be registered.

Parameters:
nameThe class name or uuid to create.
Returns:
New object or nullptr if the class wasn't found or couldn't be created.

virtual _Check_return_ void* VoodooShader::IModuleManager::FindFunction ( _In_ const String &  module,
_In_ const String &  name 
) const [pure virtual]

Finds the address of a function in a module.

The module must be loaded into the process and export the symbol, otherwise this will return nullptr. If the module name does not have an extension, ".dll" will be appended during the lookup. It is recommended you do not use an absolute path here.

Parameters:
moduleThe module name to look up.
nameThe function name to find.
Returns:
The function's address if found, nullptr otherwise.
