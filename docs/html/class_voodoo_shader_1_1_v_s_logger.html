bool VOODOO_METHODTYPE VoodooShader::VSLogger::Flush ( ) [virtual]

Immediately writes all pending data to disk.

Note:
This is useful for catchable errors which may have fatal consequences (Exception calls this in case the exception is uncaught).
Warning:
This may not (probably will not) be any good in case of a segfault or other crash. If you need complete debug logging, call Logger::SetBufferSize(int32_t) with a buffer size of 0 and all logged messages should make it to disk, even during fatal crashes.

Log a pre-formatted message.

Parameters:
levelThe log level for this message (combination of LogLevel values).
sourceThe source of the log message, usually the calling module's name.
msgThe log message.
Note:
When formatting log messages that do not need reused, it may be cleaner to do it inline: 
 pLogger->LogMessage(LL_Module | LL_Info, MODULE_NAME, Format("Hello %1%.") << "World");
This uses the Format class and implicit casts to String after the last argument is inserted.

ppOutA pointer to the new interface, passed as void**.
Returns:
