Detailed Description

Voodoo internal regex class, providing constant and compiler-safe regex handling between modules.

Voodoo internal regex match class, encapsulating matches found by Regex.

This internally uses Boost.Regex in extended-syntax mode for both expressions and match formatting.

This class provides methods to read each match group, or use them as arguments to a formatted string.
