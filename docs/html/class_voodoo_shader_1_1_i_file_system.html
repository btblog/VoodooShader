Detailed Description

Provides abstraction of the local filesystem to allow for files, multiple search directories, archives, and even network access.

Currently provides a readonly filesystem.

virtual VoodooResult VoodooShader::IFileSystem::AddPath ( const String &  path,
PathType  type = VSPath_Directory 
) [pure virtual]

Adds a directory to this file system's search path.

Parameters:
pathThe path to add (may include variables, will be parsed, may be absolute or relative).
typeThe type of the path, used to control the loading process and search handler.
Note:
This does not check for duplicates, so care should be taken not to add the same path repeatedly. Directory search order is important for file priority.
This function uses Parser::ParseString() for all paths. Variables are evaluated when the path is added.
This function will split paths at the ';' character, similar to how Windows treats the path variable. This combined with parser environment access allows regular-form environment vars to be added to the file system.
