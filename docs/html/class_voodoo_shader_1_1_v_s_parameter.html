const uint32_t VoodooShader::VSParameter::GetComponents ( ) const [virtual]

Get the component count for this parameter.

Returns:
The number of components.
Note:
This will have different meanings depending on parameter type:
  • For a texture, this represents the number of dimensions (2 for texture2D).
  • For a vector or matrix, this represents the raw number of components (2 for Float2, 4 for Float2x2, etc).
  • For an array, this represents the number of elements in the array (2 for Light[2]).
  • Structs and other types that do not also match one of the above will always return -1.

bool VoodooShader::VSParameter::QueryInterface ( Uuid &  clsid,
const void **  ppOut 
) const [virtual]

Performs type checking internally, then provides a pointer to this object with the desired interface if possible.

Parameters:
clsidThe interface Uuid desired.
ppOutA pointer to the new interface, passed as void**.
Returns:
Success of the cast or uuid-fill.
Note:
If ppOut is nullptr and clsid is a null Uuid , clsid will be filled with the this class' CLSID.
If ppOut is nullptr and clsid is not a null Uuid, this will simply check if the Uuid is allowed without performing a cast.
