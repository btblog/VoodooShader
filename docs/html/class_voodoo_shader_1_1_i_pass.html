Detailed Description

Each IPass contains a single set of programs, each operating on a different stage of the render pipeline.

Note:
A pass may contain programs for each stage. Valid stages vary by underlying API and version. In OpenGL and DirectX 9 and earlier, stages may be left to the fixed-function pipeline by not specifying any program to be used. Later APIs require the vertex and pixel shaders to be specified in each pass.
Warning:
Some adapters may require programs to be provided for certain stages.
